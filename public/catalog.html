Quick Integration

Get started with SquidPro in 3 simple steps

1. Get an API Token

curl -X POST http://localhost:8100/mint \
  -H "Content-Type: application/json" \
  -d '{"agent_id":"my-ai-agent","credits":10.0}'

2. Query Any Data Package

curl -H "Authorization: Bearer YOUR_TOKEN" \
  "http://localhost:8100/data/package/1"

3. Get Data + Receipt

{
  "trace_id": "...",
  "package_name": "Crypto Price Feed", 
  "data": {"price": 65000.50, "volume": 123.45},
  "cost": 0.005,
  "payout": {"supplier": 0.0035, "reviewer_pool": 0.001}
}
